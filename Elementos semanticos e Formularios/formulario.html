Formulário de Cadastro

Preencha os campos abaixo:

<input type="text">: Campo para entrada de texto de uma linha.
id: Identificador único para o elemento.
name: Nome do campo, usado para referenciar o dado quando o formulário é enviado.
placeholder: Dica de texto que aparece no campo antes do usuário digitar.
required: Atributo booleano que torna o preenchimento do campo obrigatório.

<input type="email">: Campo específico para endereços de email. O navegador pode validar o formato do email automaticamente.

<input type="password">: Campo de senha. Os caracteres digitados são mascarados (ex: ••••••••).
minlength="8": Exige que a senha tenha no mínimo 8 caracteres.

<select>: Cria uma lista suspensa (dropdown).
<option>: Define uma opção na lista. O atributo 'value' é o valor enviado com o formulário.

Tipo de Conta:

<input type="radio">: Permite que o usuário selecione apenas uma opção de um grupo. Todas as opções do mesmo grupo devem ter o mesmo atributo 'name'.
checked: Atributo booleano que pré-seleciona esta opção.

<input type="checkbox">: Permite que o usuário selecione uma ou mais opções. O atributo 'required' exige que a caixa seja marcada.

<textarea>: Campo para entrada de texto de múltiplas linhas.
rows: Define a altura inicial do campo em número de linhas.

<button type="submit">: Botão que, ao ser clicado, submete o formulário.
<button type="reset">: Botão que reseta os campos do formulário.
